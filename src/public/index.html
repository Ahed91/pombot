Add to Slack

Intuitive Commands

The commands are designed to match the Pomodoro Technique interactions. If at any point you forget the commands, type "pom help."

pom I will design the nav

Records your task for the next pom

pom start

Starts a pom timer for 25 minutes.

pom status

Displays time remaining in the active pom.

pom stop

Stops the active pom.

Bocoup maintains PomBot as an open source tool. To share your feedback, open an issue or a pull request on our Github repository.
